 SkillUp Academy - Landing Page

 Descripción del Proyecto

Este proyecto consiste en el diseño y desarrollo de una **Landing Page** para la startup ficticia *SkillUp Academy*, una plataforma de educación digital enfocada en el aprendizaje de habilidades tecnológicas.

El objetivo principal es presentar cursos y facilitar la navegación del usuario mediante una interfaz clara, moderna y adaptable a diferentes dispositivos.

---

 Tecnologías Utilizadas

* HTML5
* CSS3
* CSS Flexbox
* CSS Grid
* Media Queries
* Google Fonts

---

 Diseño Responsive

El sitio web fue desarrollado bajo el enfoque de diseño responsive, adaptándose a distintos tamaños de pantalla:

 Desktop: 3 columnas
 Tablet: 2 columnas
 Mobile: 1 columna

Además, el menú de navegación se adapta en dispositivos móviles mediante un **menú hamburguesa**, mejorando la experiencia de usuario.

---

 Características Principales

* Uso de **CSS Grid** para la organización del catálogo de cursos
* Implementación de **Flexbox** para la navegación y distribución de elementos
* Aplicación de **Media Queries** para la adaptación del layout
* Integración de **Google Fonts** para mejorar la tipografía
* Uso de **iconos SVG** para mantener calidad visual
* Inclusión de un **botón flotante de contacto**
